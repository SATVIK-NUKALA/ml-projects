# Diabetes Prediction using SVM

This project uses a **Support Vector Machine (SVM)** model to predict whether a person has diabetes based on medical attributes.  
It demonstrates a complete **machine learning pipeline** from data preprocessing to prediction.

---

## Features
- Data preprocessing using `StandardScaler`  
- Train-test split for evaluation  
- SVM model with linear kernel  
- Accuracy calculation (train & test)  
- Real-time prediction system  

---

## Dataset
The dataset contains the following features:

- Pregnancies  
- Glucose  
- BloodPressure  
- SkinThickness  
- Insulin  
- BMI  
- DiabetesPedigreeFunction  
- Age  

**Target Variable:**
- `Outcome` → (0 = No Diabetes, 1 = Diabetes)
